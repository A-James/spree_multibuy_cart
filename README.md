spree_multibuy_cart
=================

This extension modifies a store's cart to display all products and then only allows one to be selected at a time.

Installation
------------

Add spree_multibuy_cart to your Gemfile:

```ruby
gem 'spree_multibuy_cart'
```

Bundle your dependencies:

```shell
bundle
```
